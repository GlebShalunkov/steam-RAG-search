# 🎮 Steam Games RAG Search

**Проект:** Интеллектуальный поиск по играм Steam с использованием RAG (Retrieval-Augmented Generation)
**Дата:** 13.03.2026
**Автор:** GLEBowski
**Технологии:** Python, LangChain, ChromaDB, HuggingFace Embeddings, Pandas

## 📊 О проекте
Разработана система умного поиска по 10 000 игр из Steam. Пользователь может задавать вопросы на естественном языке, система находит наиболее релевантные игры и выдаёт их с метриками.

## 🎯 Возможности
- Поиск по смыслу (семантический поиск)
- Фильтрация по жанрам (шутеры, стратегии, гонки)
- Сортировка по популярности (отзывы, онлайн)
- Поиск бесплатных игр
- Детальная информация: разработчик, цена, онлайн

## 🔥 Ключевые результаты
- Обработано **10 000 игр**
- Самые популярные игры: **CS:GO** (7.6 млн отзывов), **PUBG** (1.5 млн), **Team Fortress 2** (1 млн)
- Найдено 130 бесплатных игр
- Максимальный онлайн: **CS:GO — 1 млн игроков**

## 🚀 Как запустить
1. Установить зависимости: `pip install langchain langchain-community chromadb pandas openpyxl`
2. Запустить ноутбук в Jupyter
3. Использовать функцию `find_shooters()` или `smart_game_answer()`

## 📈 Примеры запросов
```
find_shooters(min_positive=10000)
smart_game_answer("шутеры", min_positive=5000)
smart_game_answer("бесплатные игры", free_only=True)
```
